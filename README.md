# MacClaw Web

> **The digital home of MacClaw: High-performance automation for macOS.**

This repository contains the source code for the official MacClaw landing page and documentation site, live at **[macclaw.dev](https://macclaw.dev)**.

## 🚀 Overview

Built with:
- **React 19** + **TypeScript**
- **Vite** (Next-gen frontend tooling)
- **Tailwind CSS 4.0** (Modern styling)
- **Framer Motion** (Premium animations)
- **Lucide React** (Clean iconography)

## 🐚 Core Project

Looking for the MacClaw CLI? Check out the primary repository here:
👉 **[heisenbag/macclaw](https://github.com/heisenbag/macclaw)**

## 🛠️ Local Development

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Run dev server**:
   ```bash
   npm run dev
   ```
3. **Build for production**:
   ```bash
   npm run build
   ```
